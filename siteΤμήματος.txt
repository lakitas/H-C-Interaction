Εφαρμογή για τη σχολή Διοίκησης Επιχειρήσεων(School of Bussiness), η οποία θα έχει τους εξής στόχους:

Στόχοι:
	-> link για τη γραμματεία.
	-> θα γίνει σύνδεση με τις λειτουργείες της γραμματείας.
	-> link για το portal, ανάλογα με το ποιός έχει συνδεθεί.
	-> ημερολόγιο ανακοινώσεων(συνδεδεμένο με το portal του τμήματος).
	-> θα αποτελεί portal ενημέρωσης(links για αυτά που προσφέρει κάθε τμήμα στο aueb.gr).
	-> θα μπορεί να "ακολουθεί" έναν καθηγητή και να ενημερώνεται μέσω e-mail για νέα/ενέργειες του.

Stakeholders:
	-> φοιτητές της εν λόγω σχολής.
	-> καθηγητές της εν λόγω σχολής.
	-> άτομα που ενδιαφέρονται για πληροφορίες.
	-> μεταπτυχιακοί/τελειόφητοι.
	-> γραμματείες.
	-> άλλα πανεπιστημιακά ιδρύματα/οργανισμοί αξιολόγησης.
	-> φοιτητικοί εξωτερικοί οργανισμοί(ERASMUS, AIESEC, ...).
